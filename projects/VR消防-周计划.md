# 配电柜房火灾 — 开发计划

**交付日：2026年8月1日** | 剩余约 8 周

## 10个步骤总览

```
步骤01 准备     → 装备穿戴（绝缘手套/绝缘靴/绝缘操作杆）
步骤02 获取信息 → 联系物业电工，明确供电范围
步骤03 侦察     → 观察浓烟/柜体变形/异常声响
步骤04 断电     → 专业人员操作总闸 → 地面电流特效消失
步骤05 疏散     → 半径10米警戒区
步骤06 灭火     → 6.1 判断柜门 + 6.2 排查电缆桥架
步骤07 汇报     → 填写汇报表/口述识别
步骤08 指示     → 指挥人员指示后续工作
步骤09 隐患排除 → 通风/测温/检查线路/接头/接地
步骤10 记录     → 提交详细步骤报告
```

### 四大特效

| 特效 | 状态 | 备注 |
|---|---|---|
| 回燃火 + 明火 | ✅ 已有 | OAVA Flame + NatureManufacture |
| 流动烟雾 | 🔧 调参中 | NatureManufacture Lit_Lightmap系列 |
| 线路电火花 | ✅ 已有 | LightningBolt + GroundElectric_Random Shader |
| 阴燃 | ❌ 未实现 | 物体表面暗红微光 — 可降低优先级 |

## 周计划

### 第1周（6/2 ~ 6/8）— 粒子特效精调 🎯

| 日期 | 任务 |
|---|---|
| 周一 | Collision模块调参：火球碰墙反弹/衰减/烟雾遇天花板平铺 |
| 周二 | NatureManufacture烟雾材质适配URP |
| 周三 | 火焰+烟雾+电火花三特效联动联调 |
| 周四 | 步骤04"断电后电流消失"特效切换逻辑 |
| 周五 | 全流程特效跑通测试 + 参数固化 |

**目标：一周内特效全部到位，不再反复调。**

### 第2周（6/9 ~ 6/15）— 流程系统

- 补完所有10个步骤的 StepAsset 配置
- 确认每个步骤的分支逻辑（正确/错误路径）
- ScoutPanel + EquipmentPanel 适配所有步骤
- EnvironmentMgr 电流开关 + 灭火监听完善

### 第3周（6/16 ~ 6/22）— 核心交互

- 步骤06柜门判断逻辑（开启/关闭两条分支）
- 灭火器交互完整流程（拿起→瞄准→喷射→火焰缩小→灭火成功）
- 步骤09隐患排除的交互（测温、线路检查、接地测试）
- 步骤07汇报 UI 表单

### 第4周（6/23 ~ 6/29）— UI + 反馈

- 装备面板最终版
- 步骤反馈提示（正确/错误操作的视觉提示）
- 步骤08指挥人员指示文本对话系统
- 步骤10报告提交面板

### 第5周（6/30 ~ 7/6）— 打分系统

- 每个步骤的正确/错误权重设定
- 汇总评分逻辑
- 分数展示UI

### 第6周（7/7 ~ 7/13）— 联调 + Bug修复

- 全流程从头跑到尾
- 修复步骤间跳转的边界情况
- 特效与流程的衔接问题
- 性能优化（粒子上限/GC）

### 第7周（7/14 ~ 7/20）— 打包 + VR适配

- VR头盔适配测试（Quest/Pico/HTV）
- 输入适配（手柄交互代替鼠标点击）
- 性能档位调整（低/中/高画质）

### 第8周（7/21 ~ 8/1）— 收尾

- 最终测试
- Bug修复缓冲
- 打包交付

## 风险点

| 风险 | 影响 | 应对 |
|---|---|---|
| 阴燃特效做不出来 | 低 — 非核心 | 降级为静态发光点或直接砍掉 |
| VR手柄交互没经验 | 高 — 第7周 | 预留2周缓冲，提前调研Pico SDK |
| 打分系统逻辑复杂 | 中 | 先用简单加减分，后续迭代 |
| 性能不达标 | 中 | 第6周做Profiler，粒子设上限 |
