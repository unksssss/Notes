# VR消防项目 — 周计划

## 总体目标

完成 **高层虚拟VR仿真系统（典型业态灾害事故处置模拟系统）** 开发，共 8 个火灾场景 + 推演系统。

## 8 个场景总览

| # | 场景 | 核心难点 | 状态 |
|---|---|---|---|
| 1 | 门后回燃火 | 回燃特效、门控技术、烟气现象 | ⬜ 未开始 |
| 2 | **配电柜火** | 电火花、断电流程、干粉灭火 | 🔧 进行中 |
| 3 | 居室床火 | 烟气流动、火灾蔓延、水枪阵地 | ⬜ 未开始 |

| 5 | 承重墙虚实步 | 随机虚实墙、障碍物、蹲姿行进 | ⬜ 未开始 |
| 6 | 顶棚坍塌 | 坍塌征兆、排烟散热、人员站位 | ⬜ 未开始 |
| 7 | 物资转移 | 物资分类、路线规划、警戒标识 | ⬜ 未开始 |
| 8 | 消防电梯处置 | 电梯模式切换、门外测温、逐层判断 | ⬜ 未开始 |

## 配电柜房火灾 — 流程（当前场景）

```
步骤01 准备     → 装备穿戴（绝缘手套/绝缘靴/绝缘操作杆）
步骤02 获取信息 → 联系物业，明确配电柜供电范围
步骤03 侦察     → 观察浓烟/柜体变形/异常声响
步骤04 断电     → 专业人员操作总闸 → 地面电流特效消失
步骤05 疏散     → 半径10米警戒区
步骤06 灭火     → 6.1 判断柜门 + 6.2 排查电缆桥架
步骤07 汇报     → 填写汇报表/口述识别
步骤08 指示     → 指挥人员指示后续工作
步骤09 隐患排除 → 通风、测温、检查线路/接头/接地
步骤10 记录     → 提交详细步骤报告
```
