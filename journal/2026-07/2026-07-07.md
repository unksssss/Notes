### CharacterController 推物逻辑

**问题**：SM_凳子1 无法被 Player 碰撞推走。

**根因**：
- SM_凳子1 没有 Rigidbody（静态碰撞体）
- Player 使用 CharacterController，不会自动对 Rigidbody 施加推力

**修复**：
```cs
void OnControllerColliderHit(ControllerColliderHit hit)
{
    var rb = hit.collider.attachedRigidbody;
    if (rb == null || rb.isKinematic) return;
    Vector3 pushDir = hit.moveDirection;
    rb.AddForce(pushDir * pushForce, ForceMode.Impulse);
}
```
+ 目标物体需挂 Rigidbody（非 kinematic）。

### BreakingWallPlus — 默认参数设计

**踩坑**：`Vector3 hitPoint = new Vector3(0,0,0)` 作为默认参数 → 编译错误，不是编译时常量。

**解决方案**：
```cs
// 不传参数 → 自动用 breakingCenter.position
public void BreakAtPoint() => BreakAtPoint(breakingCenter.position);
public void BreakAtPoint(Vector3 hitPoint) { /* 精确破碎逻辑 */ }
```
重载优于可选参数判空。
