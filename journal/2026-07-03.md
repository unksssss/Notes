### 配电房项目 — CeilingCollapse.cs 天花板坍塌效果

**目标文件**: `Assets/Scripts/CeilingCollapse.cs`

实现了天花板坍塌的完整流程：碎片全部同时坠落 → 落地后播烟尘粒子 + 屏幕震动。

```cs
private IEnumerator Collapse_IE()
{
    // ① 所有碎片同时开始坠落
    foreach (var ceiling in Ceilings)
    {
        var rb = ceiling.GetComponent<Rigidbody>();
        rb.isKinematic = false;
        rb.useGravity = true;
        var bc = ceiling.GetComponent<BoxCollider>();
        bc.center = new Vector3(0, 0, 0);
        yield return null;
    }
    yield return new WaitForSeconds(1.05f);
    // ③ 全部落地 → 播烟尘特效
    ps_ceilingDust.Play();
    CameraShake.instance.StartShake();
}
```

| 配置项 | 值 | 说明 |
|--------|-----|------|
| 碎片质量 | Random.Range(0.5f, 1f) | 每块随机 |
| 碎片阻力 | Random.Range(0.5f, 1f) | 每块随机 |
| 碰撞检测 | ContinuousDynamic | 防止高速穿透 |

### 配电房项目 — CameraShake.cs 屏幕震动

**目标文件**: `Assets/Scripts/CameraShake.cs`

Singleton 模式，协程驱动，震动幅度逐帧衰减。

```cs
public static CameraShake instance;
public float duration = 0.3f;
public float intensity = 1f;

private IEnumerator Shake(float _duration, float _intensity)
{
    Vector3 originalPos = transform.position;
    float elapsed = 0f;
    while (elapsed < _duration)
    {
        float xOffset = Random.Range(-1f, 1f) * _intensity;
        float yOffset = Random.Range(-1f, 1f) * _intensity;
        transform.position = new Vector3(xOffset, yOffset) + originalPos;
        elapsed += Time.deltaTime;
        _intensity *= 0.9f;   // 每次迭代衰减 10%
        yield return null;
    }
    transform.position = originalPos;  // 复位
}
```

震动策略：
- **逐帧衰减**：`_intensity *= 0.9f`，约 30 帧减弱到初始值的 4%
- **复位保障**：协程结束后显式回到 `originalPos`，避免累积偏移
- **两轴随机**：x/y 独立随机，z 保持不动（2D 风格震动）

### ShaderGraph 踩坑 — Quad 顶点数不足导致噪声偏移无效

ShaderGraph 中用 Gradient Noise → OneMinus → Multiply(0.01) → Add(Position B) → Combine → Vertex Position，结果物体完全看不见。

**根因发现**：Quad 只有 4 个顶点，噪声采样的纹理坐标变化不足以在顶点间产生可观的位移。需要换用 Plane（更多顶点）或自定义细分网格。

修复方向：改用 10×10 分段 Plane，或使用带有足够细分的高面数模型。
