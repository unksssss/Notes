### 配电房项目 — 排烟系统 Emunition

**排烟流程**：通风口 → Wind Zone 把烟推出去 → 烟雾渐淡。

```cs
IEnumerator EjectSmoke()
{
    yield return new WaitForSeconds(5f);
    while (isCeilingSmokePlaying)
    {
        FadeOutSmoke(0.0025f);
        yield return null;
    }
    _backdraftCtrl.SetBackdraftFire(false);
}
```

### Mathf.Lerp 陷阱

```cs
// ❌ 错误：constantMax 恒不变 → 每帧算出一样的值
main.startSize = Mathf.Lerp(main.startSize.constantMax, 0f, 0.0025f);

// ✅ 正确：协程里自己计时，t 逐渐从 0 到 1
float t = elapsed / duration;
main.startSize = Mathf.Lerp(_targetSize, 0f, t);
```

Lerp 不累积——它不是 MoveTowards。渐变必须自己维护 `t`。
