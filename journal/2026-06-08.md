### 配电房项目 — 水枪 + 干粉灭火器特效收工

水枪粒子关键参数：

| 参数 | 值 | 说明 |
|---|---|---|
| Shape Angle | 3~5° | 极窄锥角，集束水流 |
| Render Mode | Stretched Billboard，Length Scale 2~4 | 粒子拉长首尾相连 |
| Start Speed | 8~15 | 水压喷射高速 |
| Gravity | 0.3~0.5 | 抛物线下垂 |
| Shader | Legacy Shaders/Particles/Additive | URP Unlit 有 AA 光晕问题 |

URP Unlit Alpha 在 Game 相机下产生 AA 边缘光晕——切回 Legacy Additive 反而效果干净。**渲染管线对透明粒子的处理差异在特效调参中优先级很高。**
