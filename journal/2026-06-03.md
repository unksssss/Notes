# 2026-06-03 星期三

## Records

### 配电房项目 — ExplosiveFireCtrl 轰燃火力控制器

新建脚本，控制轰燃场景下一键猛烈点燃所有可燃物 + 天花板烟雾扩散。

```cs
public class ExplosiveFireCtrl : MonoBehaviour
{
    [SerializeField] private ParticleSystem _ps_Ceiling;
    [SerializeField] List<FlammableObject> _flammableObjectLst;

    public void StartPlay() => StartCoroutine(Play());

    private IEnumerator Play()
    {
        // 所有物体同时猛烈点燃
        SetAllOnFire(_flammableObjectLst);
        yield return new WaitForSeconds(3f);
        _ps_Ceiling.Play();
        SetPs_Ceiling();  // 天花板烟雾变红
    }

    private void SetAllOnFire(List<FlammableObject> objs)
    {
        foreach (var obj in objs)
        {
            obj.flameParticleSize = 2f;
            obj.flameVFXMultiplier = 2f;
            obj.flameLivelinessSpeed = 2f;
            obj.ignitionTime = 0.1f;
            obj.SetOnFireFromCenter();
        }
    }
}
```

**FlammableObject 点火参数四个维度一齐拉满实现瞬间猛火：**

| 参数 | 默认 | 猛火值 | 效果 |
|---|---|---|---|
| `flameParticleSize` | 1 | 2~3 | 火焰粒子大 |
| `flameVFXMultiplier` | 1 | 2~3 | 粒子密集 |
| `flameLivelinessSpeed` | 1 | 2~3 | 跳动剧烈 |
| `ignitionTime` | 几秒 | 0.1f | 瞬间点燃 |

### 配电房项目 — NatureManufacture 烟雾 Shader 属性控制

Shader: `NM_Particles_Smoke_Lightmap_Lit`

关键运行时属性：

```cs
Material mat = smokeParticle.GetComponent<ParticleSystemRenderer>().material;

// 受光面颜色（火光照射下的亮面）
mat.SetColor("_Light_Color", new Color(1f, 0.6f, 0.3f));

// 阴影面颜色（背光面暗红）
mat.SetColor("_Shadow_Color", new Color(0.1f, 0.05f, 0.02f));

// 自发光颜色 [HDR]
mat.SetColor("_Emission_Color", new Color(0.749f, 0.055f, 0f) * 7f);

// 发光强度与对比
mat.SetFloat("_Emission_Over_Time", 1.5f);
mat.SetFloat("_Emission_Gradient_Contrast", 2f);

// 受光强度
mat.SetFloat("_Light_Intensity", 2f);
mat.SetFloat("_Light_Contrast", 1.5f);
```

### Color32 → Color 换算

```cs
// Color32 (0~255) → Color (0~1)
((Color)new Color32(191, 14, 0, 255))
// = new Color(0.749f, 0.055f, 0f, 1f)

// 直接写也行
new Color(191f / 255f, 14f / 255f, 0f);
```

### HDR Emission Intensity 的 Inspector 显示问题

`SetColor("_Emission_Color", color * 7f)` 在 Inspector 里显示的 Intensity 不是 7，而是约 2.8——Unity 对 HDR 颜色做了内部编码，Inspector 展示的是归一化后的值。**实际渲染的亮度仍然是 7 倍**，用 `GetColor` 读回来可以验证。

### ParticleSystem — StartSpeed 增大导致稀疏

增大 `Start Speed` → 粒子间距拉大 → 视觉稀疏。两条补法：

1. **加发射量**：`Emission → Rate over Time` 同比翻倍
2. **加粒子大小**：`Main → Start Size` 调大

公式：`视觉密度 ≈ (粒子数 × 粒子大小²) / 速度`
