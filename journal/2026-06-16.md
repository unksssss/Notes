# 2026-06-16 星期二

## Records

### 场景搭建知识储备

待模型到位前的准备工作梳理：

| 项目 | 知识点 |
|---|---|
| **Static 标记** | 不动的物体勾 Static → 参与 Lightmap 烘焙 + Occlusion Culling |
| **光照烘焙** | `Window → Rendering → Lighting → Generate Lighting` |
| **UV Overlap 修复** | 导入模型 → Inspector → Model → 勾 Generate Lightmap UVs |
| **烘焙清除** | `Lighting → Clear Baked Data` |
| **Occlusion Culling** | 小场景（单房间）关掉，CPU 开销 > GPU 节省 |
| **近裁面** | Camera → Clipping Planes → Near: 0.01 |

### Vefects Sand VFX 兼容修复

资源包 Shader 引用 URP 15+ 头文件，2022.3 不兼容。方案：保留 Prefab 粒子参数，材质 Shader 换成 NatureManufacture Smoke Unlit。

### CharacterController 位置覆盖

`transform.position = x` 无效 → CC 每帧覆盖。需先禁用再启用：

```cs
cc.enabled = false;
transform.position = target;
cc.enabled = true;
```

### 安全出口标识 — 有照明 vs 无照明

| | 有照明型 | 无照明/蓄光型 |
|---|---|---|
| 安装位置 | 走廊、楼梯间 | 配电房门口、机房 |
| 依赖 | 电路供电 | 自身蓄光 |
| 断电后 | 灭了 | 持续发光 |

### XR Interaction Toolkit

VR 交互核心三件套：XR Controller、XR Ray Interactor、XR Direct Interactor。后续拿灭火器、选阵地、开门都需要。

### Mathf 插值知识

- `Lerp` — 线性插值，需要自己管 `t`
- `SLerp` — 球面插值，旋转专用
- `MoveTowards` — 自动累积，适合 Update
