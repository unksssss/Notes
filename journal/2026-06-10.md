### 配电房项目 — WaterPosition 阵地选择脚本

新建水枪阵地位点脚本，玩家走进位点 → 停止铺管 → 触发回燃 → 判定阵地正确性。

```cs
public class WaterPosition : MonoBehaviour
{
    [SerializeField] private int posId;
    [SerializeField] private BackdraftCtrl backdraftCtrl;
    [SerializeField] private SingleCylinderHoseFromFireTruck fireTruck;

    private void OnTriggerEnter(Collider other)
    {
        if (!other.CompareTag("Player")) return;
        fireTruck.StopDrawing();
        backdraftCtrl.OpenDoor();
        // 后续接入 StepManager.ChooseBranchByCorrectness(stepId, isCorrect)
    }
}
```

优化建议：用 `bool _isCorrect` 替代硬编码 switch，加 `?.` 防空。

### 多路径相机系统

`MultiPathCamera`：多条不同的入口路径自动找到最近的路径 → 走完 → 汇流到公共终点段。

```cs
CameraPath FindClosestPath(cameraPos)
  → 遍历所有路径的第一个点
  → 选距离最近的
  → 先走专属路径 → 再走公共汇流段
```

适用于同一场景多个进入角度的巡览动画。
