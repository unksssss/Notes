# 2026-06-17 星期三

## Records

### XR Grab Interactable 全属性详解

`XR Grab Interactable` 是 VR 里"抓住物体跟着手走"的核心组件。

---

#### Movement（移动控制）

| 属性 | 值 | 说明 |
|---|---|---|
| **Movement Type** | `Velocity Tracking` | 物理跟随——最自然，有惯性 |
| | `Kinematic` | 直接瞬移到手位置，无物理 |
| | `Instantaneous` | 抓瞬间跳到手上，之后不跟 |
| **Smooth Position** | 勾上 | 位置过渡平滑，防抖动 ✅ |
| **Smooth Rotation** | 勾上 | 旋转过渡平滑 ✅ |
| **Velocity Scale** | 默认 1 | 跟随速度倍率。>1 甩得更远 |
| **Throw On Detach** | 勾上 | 松手时保留惯性，物体飞出去 |
| **Throw Smoothing** | 0~1 | 松手时速度平滑度 |
| **Tighten Position** | 0~1 | 抓得"紧不紧"。1=死死跟手 |
| **Tighten Rotation** | 0~1 | 旋转跟得紧不紧 |

---

#### Track（约束控制）

| 属性 | 说明 | 消防场景 |
|---|---|---|
| **Track Position** | 勾=跟手移动 | 灭火器勾，门上锁不勾 |
| **Track Rotation** | 勾=跟手旋转 | 灭火器勾，门只绕轴转不勾 |
| **Track Scale** | 不勾 | 一般不让你把灭火器捏大捏小 |

---

#### Attach Transform（抓取锚点）

| 属性 | 说明 |
|---|---|
| **Attach Transform** | 拖一个空物体 → 抓取时手"吸"到这个位置 |
| 不拖 | 默认用手碰到的那个点抓，可能歪 |

> 灭火器上放一个 Attach Transform（手柄位置）→ 抓起来手正好握在把手上。

---

#### Retain（维持控制）

| 属性 | 说明 |
|---|---|
| **Retain Transform Parent** | 被抓时保留原父级关系（门不会脱离门框） |

---

#### Interactable Events（事件系统）

| 事件 | 触发时机 | 消防用途 |
|---|---|---|
| **On Select Entered** | 被抓瞬间 | 灭火器装备到手 |
| **On Select Exited** | 松手 | 灭火器放下 |
| **On Hover Entered** | 手靠近（还没抓） | 高亮提示"可交互" |
| **On Hover Exited** | 手离开 | 取消高亮 |
| **On Activated** | 抓住状态下再按扳机 | 灭火器喷射！⭐ |
| **On Deactivated** | 松开扳机 | 停止喷射 |

---

#### Interaction Manager（交互管理器）

| 属性 | 说明 |
|---|---|
| **Interaction Manager** | 场景里唯一的 XR Interaction Manager，拖进去 |
| **Select Mode** | `Single`（一次一只手抓）/ `Multiple`（可两只手） |
| **Interaction Layer Mask** | 限制哪些手能抓。设为 Interactable 层 |

---

### XR Device Simulator 键盘映射

| 按键 | 模拟 |
|---|---|
| 鼠标左键拖拽 | 右手柄移动 |
| Shift + 鼠标左键 | 左手柄移动 |
| 鼠标中键 | 右手 Trigger（激活/点击） |
| G | Grip（抓取） |
| WASD | 头显移动 |
| 鼠标右键拖拽 | 头旋转 |
| T | 切换传送 |

### 交互组件对比

| 组件 | 物体跟手走？ | 有事件？ | 用途 |
|---|---|---|---|
| XR Simple Interactable | ❌ | ✅ 点击事件 | 按钮、阵地选择 |
| XR Grab Interactable | ✅ | ✅ 全套事件 | 灭火器、工具、门 |

### HTC 全系设备兼容

OpenXR + XRIT 原生支持 HTC Vive/Vive Pro/Cosmos/Focus 3/XR Elite。PC VR 无需额外插件，一体机需要 VIVE OpenXR Plugin。
