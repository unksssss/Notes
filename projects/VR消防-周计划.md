## 水枪阵地

```
需求：
  - 高压水柱喷射粒子（柱状/锥形水流）
  - 水流碰到物体后的溅射/反弹效果
  - 水枪可调整喷射方向和角度
  - 对火焰的降温/灭火效果

技术方案：
  - ParticleSystem Cone Shape + 细长粒子
  - Simulation Space: World
  - Collision: World 模式，溅射靠 Sub Emitter (Collision)
  - 灭火逻辑：复用 CustomParticalExtinguish 或单独水管类
```

## 干粉灭火器

```
需求：
  - 干粉喷射（雾状扩散）
  - 干粉覆盖范围可视化
  - 与 OAVA Flame 灭火联动

技术方案：
  - 已有 FoamEffect + CustomParticalExtinguish 基础
  - 改成干粉形态：粒子更小、更密、白色/灰白
  - Shape: Cone、Angle 更宽（30~45°）
  - 扩散后在地面/物体上有残留覆盖感
```
