## VR 交互清单 🔴

### 抓取类（Grab Interactable）

| 交互 | 物体 | 触发 |
|---|---|---|
| 拿灭火器 | 灭火器模型 | Grip → 装备到手，Trigger → 喷射 |
| 拿干沙子桶 | 沙子桶 | Grip → 倾倒灭火 |
| 拿验电器 | 验电器 | Grip → 靠近配电柜检测 |
| 悬挂标示牌 | "禁止合闸"牌 | Grip → 放到指定位置 |
| 装设接地线 | 接地线 | Grip → 连接配电柜 + 接地桩 |

### 点击类（Ray Interactor / UI）

| 交互 | 目标 | 触发 |
|---|---|---|
| 装备面板选择 | UI Toggle | 射线点击 |
| 获取外攻信息 | 电脑屏幕 | 点击查看 |
| 确认按钮 | UI Button | 点击推进步骤 |
| 填写汇报表 | UI 表单 | 点击输入 |
| 小地图按钮 | UI | 步骤04规划路线 |

### 操作类

| 交互 | 目标 | 触发 |
|---|---|---|
| 开配电柜门 | 柜门 | 手抓拉开关 |
| 开回燃门 | 场景门 | 手推开 |
| 扣灭火器扳机 | 灭火器 | 抓住状态下 Trigger → 喷射 |
| 瞄准喷射 | 火焰 | 手柄指向 + Trigger |

### 移动/观察

| 交互 | 方式 |
|---|---|
| 场景行走 | 连续移动 / 传送 |
| 蹲下检查配电柜底部 | 物理蹲或按钮蹲 |
| 热成像仪观察 | 手柄举起查看 |

### 阵地部署

| 交互 | 方式 |
|---|---|
| 选水枪阵地位点 | 射线指位点 + Trigger 确认 |
| 水带铺设 | 消防车触发起始位 → 走向位点 → 管道自动生成 |

| 交互 | 方式 |
