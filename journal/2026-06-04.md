# 2026-06-04 星期四

## Records

### 配电房项目 — ExplosiveFireCtrl 联调完成

轰燃流程控制器完成编码 + 场景联调测试：

```
0s   → 第0个物体点燃
3s   → 天花板烟雾开始渐变（rate↑ size↑）
8s   → 其余物体猛烈点燃 + 烟雾 Emission 变色
```

对外接口：

| 方法 | 用途 |
|---|---|
| `StartPlay()` | 开始轰燃流程（防重入） |
| `StopPlay()` | 中断流程 + 灭火（可重燃） |

防重入：`StartPlay` 内部维护 `_playRoutine`，已运行时忽略重复调用。

烟雾渐浓通过协程逐帧 `Mathf.Lerp` 实现，rate/color 平滑过渡。

### 配电房项目 — BackdraftCtrl 回燃控制器骨架

新建回燃场景控制器，核心功能：开门触发回燃粒子特效 + 门旋转动画。

```cs
public class BackdraftCtrl : MonoBehaviour
{
    [SerializeField] private ParticleSystem _ps_Backdraft;
    [SerializeField] private GameObject _door;

    public void OpenDoor()
    {
        _ps_Backdraft.Play();
        _door.transform.Rotate(new Vector3(0, 0, 90));
    }
}
```
