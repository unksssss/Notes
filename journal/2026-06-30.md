### OAVA Flame — isEffectOnForever 源码级排坑

**问题**：将 `burnOutLength_s` 设为 300 后，灭火器扑灭物体后火焰仍持续燃烧。

**根因（源码级）**：`FlammableObject.Start()` 第 297-301 行：

```cs
if (isEffectOnForever)
{
    burnOutStart_s = float.MaxValue;
    burnOutLength_s = float.MaxValue;  // ← 罪魁祸首
}
```

`isEffectOnForever = true` 不仅阻止自然 burnout，还**把 burnOutLength_s 也改成 float.MaxValue**。导致 `ExtinguishFully()` 后 burnout 动画无法在正常时间内走完（Update 中 `onFireTimer > burnOutStart_s + burnOutLength_s` 永远不满足），材质燃烧动画永远不会结束。

**正确配置（永久燃烧+灭火后正常 burnout）**：

```cs
isEffectOnForever = false;          // 不用 true！
burnOutStart_s = float.MaxValue;    // 永不自然触发 burnout
burnOutLength_s = 3f;               // 灭火后 3 秒烧完动画
isReignitable = ReIgnitable.No;     // 不复燃
```

| 参数 | isEffectOnForever=true | 手动设 MaxValue |
|---|---|---|
| burnOutLength_s | 被改为 MaxValue ❌ | 保持 3f ✅ |
| 自然 burnout | 永不触发 ✅ | 永不触发 ✅ |
| 灭火后材质动画 | 走不完 ❌ | 3 秒完成 ✅ |

### Shader — Lambert 与 Half-Lambert

- **Lambert**：`diffuse = max(0, N·L)`，背光面纯黑，过渡生硬
- **Half-Lambert**：`diffuse = N·L × 0.5 + 0.5`，[-1,1] 映射到 [0,1]，Valve 在《半条命2》引入，背光面有暗部细节，适合角色和风格化渲染
