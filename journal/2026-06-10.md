### 多路径相机系统

`MultiPathCamera`：多条不同的入口路径自动找到最近的路径 → 走完 → 汇流到公共终点段。

```cs
CameraPath FindClosestPath(cameraPos)
  → 遍历所有路径的第一个点
  → 选距离最近的
  → 先走专属路径 → 再走公共汇流段
```

适用于同一场景多个进入角度的巡览动画。

### 数字人项目 — LLM → TTS → LipSync 全管线搭建

新建完整数字人对话管线 `DigitalHumanPipeline`：

```
用户文本输入
  → LLMClient (讯飞星火 Spark Ultra-32K) → AI 回复 + 情感标签
  → WsTTSClient (WebSocket 语音合成) → 音频流
  → AudioStreamPlayer → 播放音频
  → LipSyncDriver + FaceBlendController → 驱动口型 + 表情
```

| 模块 | 文件 | 职责 |
|---|---|---|
| 主控管线 | `DigitalHumanPipeline` | 串联 LLM→TTS→LipSync |
| LLM | `LLMClient` / `SparkLLMClient` | 讯飞星火 API，HTTP 异步 |
| TTS | `WsTTSClient` | WebSocket 流式语音合成 |
| 音频 | `AudioStreamPlayer` | 播放合成音频 |
| 鉴权 | `XfyunAuth` | 讯飞 API 鉴权 |
| 输入 | `ChatInputBridge` | UI 输入桥接 |
| 测试 | `PipelineTester` | 管线调试入口 |
