# 2026-07-14 星期二

## Records

### 配电房项目 — 绳索路径追踪修复：Player 根节点 vs ropeEnd

**问题**：`Update()` 中 `delta = Distance(ropeEnd.position, _lastPlayerPos)` 在玩家不移动时 delta 仍有 ~0.65，导致绳索持续延长。

**根因**：`_lastPlayerPos` 初始化为 `_playerRoot.position`（Player 根节点）但 delta 用的 `ropeEnd.position`（喷枪端），两个点相差约 0.65 单位。加上 `_playerRoot` 通过 `ropeEnd.root` 获取，若场景引用混乱也会导致追踪错误。

**修复**：
- 统一用 `_playerRoot.position`（Player 根节点）计算 delta
- 阈值从 0.1f 改为 0.02f 让第一帧能同步
- 将 `_playerRoot` 改为 `[SerializeField]` 让用户在 Inspector 手动拖拽

```cs
// Update
float delta = Vector3.Distance(_playerRoot.position, _lastPlayerPos);
if (delta > 0.02f) {
    _totalPath += delta;
    _lastPlayerPos = _playerRoot.position;
}
```

### 配电房项目 — Player ObiRigidbody 设置修复

**问题**：玩家旋转视角时 `_playerRoot.position` 变化 ~0.4/frame，路径追踪误判为玩家移动。

**根因**：Player 的 `ObiRigidbody.kinematicForParticles = false`。Pin 约束把 `ropeEnd`（喷枪）的旋转耦合到 Player 的 Rigidbody，Obi 对其施力导致 Player 被推动。

**修复**：
- 场景中设 Player 的 `ObiRigidbody.kinematicForParticles = true`
- Player 的 `Rigidbody.useGravity = false`（CharacterController 自己处理重力）

### 配电房项目 — Alt 键视角偏移修复

**根因**：按住 Alt 时 `Cursor.lockState = Confined` 但 `_isCursorLocked` 没变 false，`HandleMouseLook` 在 Confined 模式下异常运行 + `HandleMovement` 不执行导致 Rigidbody 自动重力接管。

**修复**：按住 Alt 时设 `_isCursorLocked = false`。

### 配电房项目 — Pin 约束旋转柔度根因发现（关键）

**问题**：绳索在生成和玩家移动时持续滚动/扭曲，地面滚动摩擦无效、阻尼无效、substeps 无效。

**根因**：`ObiPinConstraintsBatch.AddConstraint` 签名中第 5、6 参数是 `linearCompliance`（位置柔度）和 `rotationalCompliance`（旋转柔度），而非 spring/damping。传 `0, 0` 意味着**位置和旋转都是完全刚性**——喷枪旋转直接被 Pin 约束 1:1 传给绳索粒子。

```cs
// 错误：旋转刚性，喷枪转绳子跟着转
batch.AddConstraint(solverIndex, body, offset, Quaternion.identity, 0, 0, infinity);

// 正确：位置刚性（粒子钉在端点），旋转自由（喷枪转绳子不滚）
batch.AddConstraint(solverIndex, body, offset, Quaternion.identity, 0, float.PositiveInfinity, infinity);
```

**修复**：将 `rotationalCompliance` 从 `0` 改为 `float.PositiveInfinity`，仅约束位置、释放旋转。

### 配电房项目 — 两版绳索方案定型

经过一天多轮迭代，确定两版并行方案并通过 Git 管理：

**版本 A：全量生成绳索**（`git checkout 全量生成绳索` / tag `全量生成绳索-v1`）
- 一次性生成 `maxRopeLength` 长的绳索，多余堆积在 AnchorCube 后
- Pin 约束 rotationalCompliance=∞ 阻断旋转传递
- 路径方向 = 水平朝锚点 + Y 向上分量（防扎穿地面）
- 无 cursor、无 LateUpdate、无碰撞材质
- Update 只做路径追踪不动绳索
- 代码 118 行，无抖动无闪烁

**版本 B：动态生成绳索**（`git checkout 绳索动态生成最终版` / tag `绳索动态生成最终版`）
- 初始只生成两端间短绳索，通过 `ObiRopeCursor.ChangeLength()` 动态延长
- 地面 `GroundFriction` 碰撞材质（rollingFriction=1）减轻滚动
- pooledParticles=500，最大绳长约 50m

**场景设置要点**：
```
Player ObiRigidbody.kinematicForParticles = true
Player Rigidbody.useGravity = false
Plane ObiCollider.CollisionMaterial = GroundFriction
```
