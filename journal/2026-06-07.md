### NatureManufacture Unlit Smoke Shader 全属性解析

Shader `NM_Particles_Smoke_Lightmap_Unlit` 所有属性按功能分解：

| 分组 | 属性 | 用途 |
|---|---|---|
| 光照方向 | `_Light_Direction`, `_Use_Scene_Light_s_Direction` | 模拟环境光方向（不是场景灯光） |
| 颜色 | `_Light_Color`, `_Shadow_Color` | 受光面/背光面颜色 |
| 强度 | `_Light_Intensity`, `_Light_Contrast`, `_Light_Blend_Intensity` | 明暗控制 |
| 透明度 | `_Alpha_Multiplier`, `_AlphaClipThreshold`, `_Intersection_Offset` | 透明度 + 深度淡出 |
| 自发光 | `_Emission_Color` [HDR], `_Emission_Gradient` | 被火焰照亮时的发光 |
| 风力 | `_Gust_Strength`, `_Shiver_Strength`, `_Bend_Strength` | 顶点动画弯曲/抖动 |
| 裁剪 | `_CullingStart`, `_CullingDistance` | 距离淡出 |

关键发现：**即使 Unlit 也受 Lightmap 纹理驱动颜色和透明度**。`_Intersection_Offset` 控制粒子靠近物体时的透明缓冲区（越大越实）。

### 水流粒子效果方案

与泡沫/干粉的粒子参数差异：

| 参数 | 水枪 | 干粉/泡沫 |
|---|---|---|
| Shape Angle | 3~5° | 15~45° |
| Render Mode | Stretched Billboard | Billboard |
| Length Scale | 2~3 | — |
| Start Speed | 8~15 | 2~4 |
| Gravity | 0.3~0.5 | 0~0.1 |

核心：窄锥角 + 拉长粒子首尾相连 → 连续水柱感。
