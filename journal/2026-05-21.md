多边形图片点击检测
```cs
using UnityEngine;  
using UnityEngine.UI;  
  
public class CustomImg : Image  
{  
    private PolygonCollider2D polygonCollider;  
    private PolygonCollider2D PolygonCollider  
    {     
get  
        {  
            if (polygonCollider == null) polygonCollider = GetComponent<PolygonCollider2D>();  
            return polygonCollider;  
        }    }    public override bool IsRaycastLocationValid(Vector2 screenPoint, Camera eventCamera)  
    {        Vector3 worldPosition;  
        RectTransformUtility.ScreenPointToWorldPointInRectangle(rectTransform, screenPoint, eventCamera,out worldPosition);  
        return PolygonCollider.OverlapPoint(worldPosition);  
    }}
```


polygonCollider:用于承接点击射线
IsRaycastLocationValid:自定义判断某个屏幕坐标是否应该被射线检测命中

利用
`RectTransformUtility.ScreenPointToWorldPointInRectangle(rectTransform, screenPoint, eventCamera,out worldPosition);`
这个方法,获取射线点击的世界坐标
再通过
`PolygonCollider.OverlapPoint`
判断点击是否在自定义碰撞体上.

最后通过IsRaycastLocationValid接口方法,返回点击的结果
